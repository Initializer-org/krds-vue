# KRDS Vue

KRDS Vue는 KRDS(Korea Digital Service Design System) 가이드를 Vue 3 + TypeScript 애플리케이션에서 사용할 수 있도록 구현한 컴포넌트 라이브러리입니다.

공공 웹서비스에서 자주 쓰는 폼, 내비게이션, 레이아웃, 피드백 UI를 일관된 API와 스타일로 제공합니다. 이 Storybook에서는 컴포넌트 예제, props, 이벤트, 접근성 상태를 함께 확인할 수 있습니다.

> 패키지는 ESM 전용입니다. CommonJS `require()`는 지원하지 않습니다.

## 먼저 볼 것

- **컴포넌트 예제**: 왼쪽 사이드바에서 컴포넌트를 선택하면 기본 사용 예제를 볼 수 있습니다.
- **Controls**: Story 화면의 Controls 패널에서 props를 바꿔 상태를 확인합니다.
- **Docs**: 컴포넌트별 문서 탭에서 props, events, slot 사용 방식을 확인합니다.
- **Theme toolbar**: 상단 Theme 도구로 `light`와 `high-contrast` 모드를 전환합니다.

## 설치

```bash
pnpm add @krds.ui/vue
```

```ts
import '@krds.ui/vue/style'
```

## 전역 등록

```ts
import { createApp } from 'vue'
import KrdsVue from '@krds.ui/vue'
import '@krds.ui/vue/style'
import App from './App.vue'

createApp(App).use(KrdsVue).mount('#app')
```

특정 컴포넌트만 전역 등록할 수도 있습니다.

```ts
app.use(KrdsVue, {
  components: ['KrdsButton', 'KrdsInput', 'KrdsModal']
})
```

## 개별 import

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { KrdsButton, KrdsInput } from '@krds.ui/vue'

const name = ref('')
</script>

<template>
  <KrdsInput v-model="name" label="이름" required />
  <KrdsButton variant="primary">확인</KrdsButton>
</template>
```

## 스타일과 테마

KRDS Vue는 CSS custom properties 기반의 디자인 토큰을 사용합니다. 토큰을 조정해야 한다면 원본 파일을 수정하지 말고 애플리케이션 CSS에서 필요한 변수만 재선언합니다.

```css
:root {
  --krds-color-light-primary-50: #256ef4;
}
```

컬러 모드는 `html` 또는 루트 요소의 `data-krds-mode` 속성으로 전환합니다.

```ts
document.documentElement.setAttribute('data-krds-mode', 'high-contrast')
```

지원 모드는 다음과 같습니다.

- `light`: 기본 라이트 모드
- `high-contrast`: 고대비 모드
- `theme`: 사용자 시스템 설정에 따라 라이트/고대비 스타일 적용

## 컴포넌트 범위

- **Actions**: Button, ButtonGroup, Link, Tag, Badge
- **Forms**: Input, Textarea, Select, DateInput, Checkbox, Radio, ToggleSwitch, FileUpload
- **Navigation**: Breadcrumb, Pagination, SideNavigation, InPageNavigation, StepIndicator
- **Layout**: Layout, Header, Footer, Masthead, Identifier
- **Feedback/Overlay**: Modal, Tooltip, CoachMark, ContextualHelp, CriticalAlerts, Spinner
- **Data/Content**: Table, StructuredList, TextList, Panel, Disclosure
